# WeiboSuperScrapy
最强微博爬虫，用户、话题、评论一网打尽。

## GUI 功能集中版

运行 GUI.py 即可爬取用户/话题微博

运行 WeiboCommentScrapy.py 并修改里面的微博id (wid) 即可爬取指定微博的所有评论。

## 无 GUI 功能独立版

单独的 py 文件分别对立一个 功能

WeiboCommentScrapy.py 爬取评论

WeiboTopicScrapy.py   爬取指定关键词的所有微博，突破了 50 页的限制，可指定截至日期
